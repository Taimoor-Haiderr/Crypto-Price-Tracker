# Crypto-Price-Tracker
A modern Crypto Price Tracker desktop application built with Python and Tkinter. It provides real-time cryptocurrency prices, portfolio tracking, price alerts, historical data visualization, and interactive charts using the CoinGecko API.

A powerful and visually modern desktop cryptocurrency tracking application developed using Python. This app allows users to monitor live crypto prices, manage portfolios, set alerts, and analyze historical trends — all in one place.

## Features
## Live Market Data
Real-time prices using CoinGecko API
24h price change indicators
Market cap & trading volume
Dynamic sparkline charts

## Search Cryptocurrency
Search coins by name or symbol
Add coins to watchlist instantly

## Portfolio Tracker
Track your crypto holdings
Calculate profit/loss in real-time
Total portfolio value overview

## Price Alerts
Set custom price alerts
Get notified when target price is reached

## Price History & Charts
View historical data logs
Interactive 7-day mini charts
CSV-based data storage

## Modern UI
Clean dark theme interface
Smooth navigation and responsive layout

## Technologies Used
Python
Tkinter (GUI)
Requests (API calls)
JSON & CSV (Data storage)
CoinGecko API

## Installation

## Usage
Launch the app
View live crypto prices
Add coins to your watchlist
Track your portfolio
Set alerts for price changes

## Data Storage
crypto_history.csv → price history
